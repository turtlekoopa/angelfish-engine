# angelfish engine
A free and open source game engine made using the raylib library inspired by the source game engine 

# compatibility
as of now angelfish has only been tested on macos and linux however it possible to make angelfish work on other systems but would require manual setup 

# features
main game loop
sceneloader
logger class

# NOTICE
project angelfish is in indev (in development) this release is just a engine skeleton where more is gonna get added on later
also angelfish main build is now called angelfish core however new modified versions or templates are planned to be built and have 2d support

# Credits
huge thanks for the libary raylib for making a window easy their website -> raylib.com

# requirements
1. the raylib libary denpendencies examples: .so, .a, .dylib, .dll, .lib
2. A pc with opengl version 1.1+ or ES 2.0+ support
3. A compilier configered for raylib
