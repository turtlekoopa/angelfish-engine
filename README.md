# Angel-fish-engine
A free and open source game made using the raylib library inspired by the source game engine 

# compatibility
as of now angel fish has only been tested on macos however it possible to make angel fish work on other systems but would require manual setup

# features
main game loop
player texture (no deticated texture engine planed to be written soon)
simple topdown player movement

# NOTICE
project angel fish is in indev (in development) this release is just a engine skeleton where more is gonna get added on later
