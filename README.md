# angelfish engine
A free and open source game engine made using the raylib library inspired by the source game engine 

# compatibility
as of now angelfish has only been tested on macos however it possible to make angelfish work on other systems but would require manual setup

# features
main game loop
player texture (no deticated texture engine planed to be written soon)
simple topdown player movement

# NOTICE
project angelfish is in indev (in development) this release is just a engine skeleton where more is gonna get added on later

1. the raylib libary denpendencies examples: .so, .a, .dylib, .dll, .lib
